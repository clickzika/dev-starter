# devstarter-config-sync.md — Sync devstarter-config.yml → .project.env

## Model: Haiku (`claude-haiku-4-5-20251001`)

## Purpose

`devstarter-config.yml` is the primary config file. `.project.env` is a
generated compatibility layer for bash scripts that cannot parse YAML.

Run this sync whenever you edit `devstarter-config.yml`.

---

## When to Run

- After editing `devstarter-config.yml` (VCS, PM, CI, AI changes)
- After cloning a project that has `devstarter-config.yml` but no `.project.env`
- When a workflow fails because `.project.env` is out of date

---

## Sync Procedure

### Method 1 — Python (recommended, no extra tools needed)

```bash
python3 << 'SYNCEOF'
import yaml, os, sys

config_path = "devstarter-config.yml"
output_path = ".project.env"

if not os.path.exists(config_path):
    print(f"ERROR: {config_path} not found")
    sys.exit(1)

with open(config_path, "r") as f:
    c = yaml.safe_load(f)

proj  = c.get("project", {})
vcs   = c.get("vcs", {})
pm    = c.get("pm", {})
ci    = c.get("ci", {})
sec   = c.get("secrets", {})
ai    = c.get("ai", {})
stack = c.get("stack", {})

lines = [
    "# .project.env — AUTO-GENERATED from devstarter-config.yml",
    "# Do not edit this file directly. Edit devstarter-config.yml instead.",
    "# Regenerate: python3 -c \"$(cat sdlc/devstarter-config-sync.md | grep -A50 'python3 <<')\"",
    "",
    "# PROJECT",
    f"PROJECT_TYPE={proj.get('type', 'app')}",
    f"LANGUAGE={proj.get('language', '')}",
    "",
    "# VERSION CONTROL",
    f"VCS_TYPE={vcs.get('type', 'github')}",
]

vcs_type = vcs.get("type", "github")
if vcs_type == "github":
    lines.append(f"GITHUB_REPO={vcs.get('repo', '')}")
elif vcs_type == "gitlab":
    lines.append(f"GITLAB_URL={vcs.get('gitlab_url', 'https://gitlab.com')}")
    lines.append(f"GITLAB_PROJECT={vcs.get('gitlab_project', '')}")
elif vcs_type == "svn":
    lines.append(f"SVN_URL={vcs.get('svn_url', '')}")
    lines.append(f"SVN_TRUNK={vcs.get('svn_trunk', 'trunk')}")
    lines.append(f"SVN_BRANCHES={vcs.get('svn_branches', 'branches')}")
    lines.append(f"SVN_TAGS={vcs.get('svn_tags', 'tags')}")

lines += [
    f"VCS_SECONDARY_1={vcs.get('secondary_1', 'none')}",
    f"VCS_SECONDARY_2={vcs.get('secondary_2', 'none')}",
    f"VCS_SYNC_BRANCHES={vcs.get('sync_branches', 'main develop')}",
    "",
    "# BRANCHING STRATEGY",
    f"BRANCH_STRATEGY={vcs.get('branch_strategy', 'gitflow')}",
    f"MAIN_BRANCH={vcs.get('main_branch', 'main')}",
    f"DEV_BRANCH={vcs.get('dev_branch', 'develop')}",
    "",
    "# PROJECT MANAGEMENT",
    f"PM_TYPE={pm.get('type', 'none')}",
]

pm_type = pm.get("type", "none")
if pm_type == "notion":
    lines.append(f"NOTION_DATABASE_ID={pm.get('notion_database_id', '')}")
    lines.append(f"NOTION_BOARD_URL={pm.get('notion_board_url', '')}")
elif pm_type == "jira":
    lines.append(f"JIRA_URL={pm.get('jira_url', '')}")
    lines.append(f"JIRA_PROJECT={pm.get('jira_project', '')}")
    lines.append(f"JIRA_BOARD_ID={pm.get('jira_board_id', '')}")
    lines.append(f"JIRA_DEFAULT_ISSUE_TYPE={pm.get('jira_default_issue_type', 'Story')}")

lines += [
    "",
    "# CI/CD",
    f"CI_TYPE={ci.get('type', 'github-actions')}",
    "",
    "# SECRETS BACKEND",
    f"SECRETS_BACKEND={sec.get('backend', 'env-file')}",
    "",
    "# AI PROVIDER",
    f"AI_PROVIDER={ai.get('provider', 'claude')}",
]

ai_provider = ai.get("provider", "claude")
if ai_provider == "litellm":
    lines.append(f"AI_PROVIDER_URL={ai.get('provider_url', 'http://localhost:4000')}")

if stack:
    lines += [
        "",
        "# TECH STACK",
        f"FRONTEND={stack.get('frontend', '')}",
        f"BACKEND={stack.get('backend', '')}",
        f"DATABASE={stack.get('database', '')}",
    ]

with open(output_path, "w") as f:
    f.write("\n".join(lines) + "\n")

print(f"✅ Synced: {config_path} → {output_path}")
SYNCEOF
```

### Method 2 — yq (if installed)

```bash

# Then run Method 1 for full sync
```

## Verify Sync

```bash
# Check values match
cat .project.env
grep "vcs:" devstarter-config.yml
```

---

## When an Agent Reads Config

**Agents (in runbooks):** Read `devstarter-config.yml` directly — use YAML paths like `vcs.type`, `pm.type`

**Bash scripts (in runbooks):** Source `.project.env` — run sync first if stale:
```bash
source .project.env 2>/dev/null || true
echo "VCS: $VCS_TYPE | PM: $PM_TYPE | CI: $CI_TYPE"
```

---

## Keeping in Sync

Add this to your project's Git pre-commit hook to auto-sync:

```bash
# .git/hooks/pre-commit
if git diff --cached --name-only | grep -q "devstarter-config.yml"; then
  echo "devstarter-config.yml changed — syncing .project.env..."
  python3 sdlc/devstarter-config-sync.md  # or extract the script above
fi
```
