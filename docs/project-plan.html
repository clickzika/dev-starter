Roadmap

Original roadmap superseded

The v1.5.0–v2.0.0 roadmap in earlier revisions of this plan was superseded — those version numbers shipped with different scope than originally planned (see Release History). The toolkit is now at v5.9.0.

Forward roadmap is tracked in GitHub Issues + milestones on clickzika/dev-starter-dev (PM moved from Notion to GitHub Issues). Original planned themes that did land in some form: automated validation/CI (v3.6.0 fitness functions), broader VCS support (v1.2.0 GitLab/SVN), and multi-agent workflows (v4.5.0 verification-loop/council).

Definition of Done

A feature or fix is "Done" when ALL of the following are true:

  • The SDLC runbook (sdlc/devstarter-X.md) is complete and follows all workflow rules
  • The slash command file (commands/devstarter-X.md) exists and routes correctly
  • The command is registered in devstarter-menu.md
  • All internal cross-references use the devstarter- prefix
  • VERSION file is bumped appropriately (SemVer)
  • CHANGELOG.md has a new entry documenting the change
  • Pre-release regression checklist passes (all 8 checks)
  • Feature branch is merged to develop via PR
  • Release flow: develop → main → tag → push to release remote
  • Notion task status updated to "Done"
