Overviewภาพรวม

DevStarter v6.0.0 ships the Obsidian Vault Design + Ingest Skill — a major addition to the knowledge management layer. Users can now organize vault notes in a structured hierarchy and ingest any existing Markdown file into the vault with auto-classification and wikilink generation. DevStarter v6.0.0 นำเสนอ Obsidian Vault Design + Ingest Skill — การปรับปรุงใหญ่ในชั้น knowledge management ผู้ใช้สามารถจัดระเบียบโน้ตในโฟลเดอร์แบบมีโครงสร้าง และนำไฟล์ Markdown ที่มีอยู่เข้า vault พร้อมการจัดประเภทอัตโนมัติและสร้าง wikilink
