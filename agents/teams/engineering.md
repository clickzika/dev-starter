# Team: Engineering

This team builds and implements features across all platforms, including database design.

## Members
- **Backend** → `~/.claude/agents/backend.md` — APIs, services, server-side logic
- **Frontend** → `~/.claude/agents/frontend.md` — UI, components, client-side logic
- **Mobile** → `~/.claude/agents/mobile.md` — Mobile apps, responsive design, native features
- **DBA** → `~/.claude/agents/dba.md` — Database design, schema, migrations, query optimization

## When to use
- Feature implementation (full-stack or platform-specific)
- Bug fixes in application code
- API design and integration
- UI/UX implementation
- Database schema design and migrations
- Query optimization and performance tuning

## How to invoke
Use `@team-engineering` or call individual agents:
- `@backend` — APIs & server-side
- `@frontend` — UI & client-side
- `@mobile` — mobile apps
- `@dba` — databases & queries
