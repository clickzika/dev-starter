# Team: Leadership

This team handles strategy, planning, requirements, and architecture decisions.

## Members
- **Tech Lead** → `~/.claude/agents/devstarter-techlead.md` — Owns architecture, code review, ADRs, system design
- **PM** → `~/.claude/agents/devstarter-pm.md` — Product roadmap, sprint planning, stakeholder alignment
- **BA** → `~/.claude/agents/devstarter-ba.md` — Requirements gathering, user stories, acceptance criteria

## When to use
- Project kickoff and planning
- Architecture decisions and trade-offs
- Requirements analysis and prioritization
- Sprint planning and roadmap reviews

## How to invoke
Use `@team-leadership` or call individual agents:
- `@devstarter-techlead` — architecture & code review
- `@devstarter-pm` — product & planning
- `@devstarter-ba` — requirements & analysis
