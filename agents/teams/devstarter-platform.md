# Team: Platform

This team manages infrastructure, deployments, CI/CD, and security.

## Members
- **DevOps** → `~/.claude/agents/devstarter-devops.md` — CI/CD, Docker, cloud infrastructure, IaC, deployments
- **Security** → `~/.claude/agents/devstarter-security.md` — Security audits, vulnerability assessment, compliance

## When to use
- CI/CD pipeline setup or troubleshooting
- Infrastructure provisioning and scaling
- Docker and container orchestration
- Security audits and vulnerability fixes
- Deployment automation
- Cloud architecture and cost optimization

## How to invoke
Use `@team-platform` or call individual agents:
- `@devstarter-devops` — CI/CD, infra & automation
- `@devstarter-security` — security & compliance
