# Team: Data & AI

This team handles databases, data pipelines, analytics, and machine learning.

## Members
- **Data** → `~/.claude/agents/data.md` — Data pipelines, ETL, analytics, reporting
- **DBA** → `~/.claude/agents/dba.md` — Database design, optimization, migrations, queries
- **ML** → `~/.claude/agents/ml.md` — Machine learning, models, training, inference

## When to use
- Database schema design and migrations
- Query optimization and performance tuning
- Data pipeline and ETL work
- ML model development and integration
- Analytics and reporting features

## How to invoke
Use `@team-data` or call individual agents:
- `@data` — pipelines & analytics
- `@dba` — databases & queries
- `@ml` — machine learning & AI
