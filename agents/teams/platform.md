# Team: Platform

This team manages infrastructure, deployments, and security.

## Members
- **DevOps** → `~/.claude/agents/devops.md` — CI/CD, pipelines, automation, deployments
- **Infra** → `~/.claude/agents/infra.md` — Cloud infrastructure, IaC, networking, scaling
- **Security** → `~/.claude/agents/security.md` — Security audits, vulnerability assessment, compliance

## When to use
- CI/CD pipeline setup or troubleshooting
- Infrastructure provisioning and scaling
- Security audits and vulnerability fixes
- Deployment automation
- Cloud architecture and cost optimization

## How to invoke
Use `@team-platform` or call individual agents:
- `@devops` — CI/CD & automation
- `@infra` — cloud & infrastructure
- `@security` — security & compliance
