# Team: Quality & Delivery

This team ensures quality, documentation, and user experience.

## Members
- **QA** → `~/.claude/agents/qa.md` — Testing strategy, test cases, automation, bug verification
- **Docs** → `~/.claude/agents/docs.md` — Technical documentation, API docs, guides
- **UX/UI** → `~/.claude/agents/uxui.md` — User experience, design review, accessibility

## When to use
- Test planning and writing test cases
- Documentation creation and updates
- UX/UI review and accessibility audits
- Pre-release quality checks
- User-facing content review

## How to invoke
Use `@team-quality` or call individual agents:
- `@qa` — testing & quality
- `@docs` — documentation
- `@uxui` — design & accessibility
