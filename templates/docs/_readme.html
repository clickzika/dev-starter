What Is This Portal?

This is a single-page viewer for all project documents produced during Gate 1 (Document Phase). Instead of opening each HTML file individually, you can browse everything from one place.

How to Use

Viewing Documents

Click any document in the left sidebar. It loads in the main workspace area on the right. You can resize the sidebar by dragging the border between the sidebar and the workspace.

Bookmarking

When you click a document, the URL hash updates (e.g. #api-reference.html). You can bookmark or share this URL to jump directly to a specific document.

Opening the Portal

Open docs/index.html in your browser. That's it — no server needed, everything runs locally.

For Agents — How to Register New Documents

Rule: Every time you create a new HTML doc in docs/, you MUST also update docs/index.html Add your document to the DOCUMENTS array in the <script> section, under the appropriate section.

Find the DOCUMENTS array in docs/index.html and add your entry:

// Example: adding a new document
{
  section: "Technical Design",    // existing section name, or create new one
  docs: [
    ...existing docs...
    { name: "Your Document Title",  file: "your-doc.html",  color: "var(--cat-other)" },
  ]
}
