# DevStarter Context Templates

Paste the relevant section into your CLAUDE.md (or mention it in chat) to switch Claude's behavior mode.

| File | When to use |
|------|-------------|
| `dev.md` | Active coding/implementation sessions |
| `research.md` | Investigation, architecture exploration, learning |
| `review.md` | PR review, code audit, security pass |

## Usage

**Option A — Per session (chat)**
Copy-paste the file content into your first message.

**Option B — Project CLAUDE.md**
Add the relevant context block to your project's CLAUDE.md under a `## Session Mode` section.

**Option C — Inline reference**
Tell Claude: "Use research context" — it will adjust behavior accordingly.
